| **Factor de riesgo** | **Nivel definido** |
| -: | --- |
| Obesidad abdominal: | Circunferencia de cintura |
| Hombres | `>` 102cm |
| Mujeres | `>` 88cm |
| Triglicéridos: | `>`=150 mg/dL |
| Colesterol-DHL: | |
| Hombres | <40 mg/dL |
| Mujeres | <50 mg/dL |
| Presión arterial: | `>`=130/80 mm Hg |
| Grucosa en ayunas: | `>`=110 mg/dL |
